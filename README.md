# house-parsing-project
В этом репозитории находится модуль, при вызове которого можно извлечь в виде нескольких таблиц паспорт дома с сайта www.reformagkh.ru.


## Использование и пример

Чтобы воспользоваться модулем, необходимо импортировать его в вашу программу:

```Python
import get_house
```

Затем, нужно применить функцию из модуля к ссылке на страницу дома на сайте, например, такую:

```Python
link = 'https://www.reformagkh.ru/myhouse/profile/view/8034988/'
get_house.get_passport(link)
```
